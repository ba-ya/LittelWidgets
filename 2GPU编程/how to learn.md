| 特性         | 普通 C++ 编程          | GPU 编程（CUDA）                                             | 学习重点                         |
| ------------ | ---------------------- | ------------------------------------------------------------ | -------------------------------- |
| **计算单元** | CPU，少量核心          | GPU，成百上千核心                                            | 理解 CPU vs GPU 架构差异         |
| **并行模式** | 串行为主，可多线程     | 并行为主，SIMT（Single Instruction, Multiple Threads）       | 学会拆分问题为成千上万线程       |
| **内存模型** | 单一内存空间，统一访问 | 分层内存：寄存器（最快）、共享内存、全局内存（慢）、常量/纹理内存 | 理解内存层次和访问速度差异       |
| **编程语言** | 标准 C++               | CUDA C++（基于 C++） + kernel 函数                           | 学会 kernel 的写法和线程索引管理 |
| **调试难度** | 普通调试即可           | 多线程调试复杂，错误可能只在部分线程出现                     | 学会使用 CUDA 调试工具和分析性能 |
| **优化方式** | 算法优化 + 编译器优化  | 并行布局 + 内存优化 + 分支减少 + 硬件特性利用                | 掌握并行算法设计与内存访问优化   |
| **性能瓶颈** | 算法复杂度             | 内存访问、线程利用率、分支/同步开销                          | 学会分析瓶颈并优化 kernel        |
| **适合问题** | 逻辑复杂、顺序任务     | 大规模数据并行计算，如矩阵运算、图像处理                     | 找到适合并行化的问题去练手       |
| **学习难度** | 初学容易               | 初学稍难，但有规律                                           | 从小例子开始，循序渐进           |

# 路线

## **阶段 1：GPU & 并行计算基础**

**目标**：理解 GPU 架构、并行计算模型、线程、块、网格和内存层次结构。

**书籍/资源**：

1. **《Programming Massively Parallel Processors》前几章**
   - 重点理解 GPU 架构、线程块、网格、内存层次（register/shared/global memory）。
2. **《GPU Gems》第一卷**
   - 可以看到 GPU 并行计算在实际中的应用。

**练习**：

- 理解 CPU vs GPU 的区别
- 写一些理论题：计算线程数、块数、网格布局
- 思考如何将数组计算并行化

------

## **阶段 2：CUDA 入门**

**目标**：能写出简单 CUDA 程序，理解 kernel、thread、block、memory 的使用。

**书籍/资源**：

1. **《CUDA by Example》**
   - 核心章节：
     - Chapter 1-3：CUDA 编程入门，Hello World, vector addition
     - Chapter 4-6：线程块、内存优化
     - Chapter 7-8：矩阵运算、并行算法
2. **NVIDIA 官方 CUDA 教程**
   - 官方提供的 sample code 可以直接运行

**练习**：

- 实现向量加法、向量缩放
- 实现矩阵乘法
- 修改线程块大小，观察性能变化
- 尝试使用 shared memory 优化程序

------

## **阶段 3：CUDA 深入**

**目标**：理解性能优化、内存管理、常用库（cuBLAS、cuFFT、Thrust）。

**书籍/资源**：

1. **《Programming Massively Parallel Processors》后半部分**
   - 高级优化策略，内存访问模式优化
2. **NVIDIA CUDA Toolkit 文档**
   - 学习 cuBLAS/cuFFT/Thrust 等库的用法

**练习**：

- 使用 cuBLAS 做矩阵乘法
- 使用 Thrust 做排序、扫描（scan）操作
- 尝试写优化后的矩阵乘法，比较全局内存 vs shared memory

------

## **阶段 4：实战项目**

**目标**：结合算法和 CUDA，做一些完整项目。

**项目建议**：

- 图像处理：滤波、卷积、边缘检测
- 数值计算：大规模矩阵运算、微分方程模拟
- 深度学习加速：用 CUDA 实现简单的前向/反向传播

**练习方式**：

- 先用 CPU 实现算法
- 分析计算瓶颈
- 用 CUDA 改写性能关键部分
