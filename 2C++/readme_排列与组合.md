------

## 🌟 一句话区分

> - **排列（Permutation）**：**考虑顺序** 的选法  
> - **组合（Combination）**：**不考虑顺序** 的选法

------

## ✅ 一、排列数（Permutation）

### 🔹 定义：

从 `n` 个不同元素中，**选出 `m` 个，并按一定顺序排成一列**，有多少种排法？

### 🔹 公式：

$ P(n, m) = A_n^m = n \times (n-1) \times (n-2) \times \cdots \times (n-m+1) = \frac{n!}{(n - m)!} $

### 🔹 例子：

从 3 个人（A、B、C）中选 2 人 **排队拍照**（有前后顺序）：

可能的排法：

- A B
- B A
- A C
- C A
- B C
- C B

✅ 共 **6 种**

计算：
$ P(3, 2) = 3 \times 2 = 6 $

> 👉 因为“AB”和“BA”是不同的（顺序不同），所以算两种。

------

## ✅ 二、组合数（Combination）

### 🔹 定义：

从 `n` 个不同元素中，**选出 `m` 个，不考虑顺序**，有多少种选法？

### 🔹 公式：

$ C(n, m) = C_n^m =\binom{n}{m} = \frac{n!}{m! \cdot (n - m)!} $

### 🔹 例子：

从 3 个人（A、B、C）中选 2 人 **组成一个小组**（没有顺序）：

可能的组合：

- {A, B}
- {A, C}
- {B, C}

✅ 共 **3 种**

计算：
$ C(3, 2) = \frac{3 \times 2}{2 \times 1} = 3 $

> 👉 因为“AB”和“BA”是同一个小组，只算一种。

------

## 🔁 两者关系

$ P(n, m) = C(n, m) \times m! $

解释：
先选出 `m` 个元素（组合），再把它们全排列（`m!` 种顺序）→ 得到排列数。

------

## 📊 对比表

| 项目           | 排列数 ( P(n,m) )      | 组合数 ( C(n,m) )        |
| -------------- | ---------------------- | ------------------------ |
| 是否考虑顺序？ | ✅ 是                   | ❌ 否                     |
| 应用场景       | 排队、密码、排名       | 抽奖、组队、选课         |
| 公式           | $\dfrac{n!}{(n-m)!} $  | $ \dfrac{n!}{m!(n-m)!} $ |
| 大小比较       | $ P(n,m) \geq C(n,m) $ | 更小                     |

------

## 💡 编程中的常见用法

### C++（C++20 起支持）

```cpp
#include <numeric>
auto p = std::permuted_count(n, m); // 没有直接函数，需自己算
auto c = std::combinations(n, m);   // 也没有，但可用公式
```

通常手动计算（注意溢出）：

```cpp
// 计算 C(n, k)
long long comb(int n, int k) {
    if (k > n - k) k = n - k; // 利用对称性
    long long res = 1;
    for (int i = 0; i < k; i++) {
        res = res * (n - i) / (i + 1);
    }
    return res;
}
```

### Python

```python
import math
from math import comb, perm

print(perm(3, 2))  # 6
print(comb(3, 2))  # 3
```

## 🎯 小技巧：如何判断用排列还是组合？

问自己一个问题：

> **交换两个元素的位置，结果会变吗？**

- 会变 → 用 **排列**（如密码 123 ≠ 321）
- 不会变 → 用 **组合**（如选苹果和香蕉，和选香蕉和苹果是一样的）

## ✅ 总结

| 概念     | 关键词 | 公式                            | 例子       |
| -------- | ------ | ------------------------------- | ---------- |
| **排列** | 有序   | $ P(n,m) = \frac{n!}{(n-m)!} $  | 排队、密码 |
| **组合** | 无序   | $C(n,m) = \frac{n!}{m!(n-m)!} $ | 抽奖、组队 |

------

🎯 **记住**：

> **排列讲顺序，组合讲搭配**。
> 顺序重要用排列，顺序无关用组合！
